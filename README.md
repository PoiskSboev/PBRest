# PBRest

## Getting Started

### Rerequisites:
* Nodejs
* npm install -g http-server

### Clone PBRest 
In a new terminal window
```
git clone https://github.com/SafeCodee/PBRest.git
cd PBRest
```
### Run servers
```
npm install
node dev
```

Got to [http://localhost:3002](http://localhost:3002)
